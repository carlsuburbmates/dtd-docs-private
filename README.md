# DTD Documentation Repository (Private)

This is the private documentation repository for the **dogtrainersdirectory.com.au** project.

## Contents

- **`.github/copilot-instructions.md`** – Copilot/AI agent onboarding: architecture, workflow, verification, critical rules, and integration points.
- **`assessment/`** – Baseline reports, implementation assessments, and audit artifacts.
- **`supabase/`** – Database setup and configuration documentation.

## Access Control

This repository is **private**. Only authorized contributors have access.

## Related Repositories

- **`dtd-app`** – The public code repository (Next.js app, API, migrations, tests).
- This repo – Private docs, runbooks, evidence, and operational artifacts.

## Setup for Contributors

Clone both repositories side-by-side:

```bash
git clone git@github.com:carlsuburbmates/dtd-app.git
git clone git@github.com:carlsuburbmates/dtd-docs-private.git
```

Then refer to the **`dtd-docs-private/` instructions in conjunction with **`dtd-app/`** during development and deployment.
